# Chapter 5 Working With Text
การทำงานกับข้อความในlinux เกี่ยวข้องกับการจัดข้อมูลเพื่อเอาข้อมูลมาใช้หรือแปลงเป็นเอาต์พุตที่ต้องการ ในlinuxทุกอย่างถือว่าเป็นไฟล์รวมถึงข้อความด้วย การที่เรามีพื้นฐานการใช้คำสั่งต่างๆที่เกี่ยวกับการประมวลผลช้อความในlinuxทำให้ สามารถทำงานเสร็จได้และมีประสิทธิภาพอย่างมากและช่วยประหยัดเวลาในการแก้ไขปัญหาจากงานที่ยากและน่าเบื่อสามารถแก้ไขได้ภายในไม่กี่วินาที
